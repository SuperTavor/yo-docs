# Character IDs

## Format

### First Character

**YKW1:**

|Letter|Character Type|
|------|--------------|
|c     |Humans|
|d     |Vehicles|
|i     |Interactable Objects|
|m     |Animated Non-interactable Objects|
|r     |Catchable animals|
|x     |Yo-kai Tribe Medals|
|y     |Yo-kai|
|z     |*???*|
